BFC文档->网络组件->下载管理器

 

 

组件名：下载管理器

英文名：DownloadManager      Jar名称：BFC_Component_DownloadManager_VX.XX.jar

 

实现需求：

为应用程序提供单独的下载服务与调用接口，实现添加下载、删掉下载、状态查询、下载状态回调等应用接口，

目前只支持单个任务下载。

 

功能说明：

支持HTTP方式下载，功能实现包括：

l  单个下载任务添加、删除、暂停。

l  多个下载任务添加、删除、暂停。

l  支持步步高数据下载并进行校验处理。

基本用法：

1、下载相关jar包文件放于目标工程的libs中 ：BFC_Utils.jar（下载） 、 gson-2.2.2.jar（下载）、MD5.jar（用到MD5校验必须要有 下载）

2、添加android 使用权限，修改AndroidManifest.xml 文件如下：

    <!-- Download System Start -->

    <uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED" />

    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

    <uses-permission android:name="android.permission.INTERNET" />

    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

    <uses-permission android:name="android.permission.WAKE_LOCK" />

    <uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />

    <uses-permission android:name="android.permission.CHANGE_WIFI_STATE" />

    <uses-permission android:name="android.permission.CHANGE_NETWORK_STATE" />

    <!-- Download System End -->

 

3、添加下载相关的组件，下载服务、下载广播监听器等，在AndroidManifest.xml 文件添加如下：

<!-- Download System Start -->

     <provider

       android:name="com.eebbk.bfc.component.download.provider.DownloadProvider"

      android:authorities="你的应用包名.download">

      <path-permission

      android:pathPrefix="/my_downloads"

      android:permission="android.permission.INTERNET" />

      <path-permission

      android:pathPrefix="/all_downloads"

      android:permission="android.permission.INTERNET" />

      <path-permission

      android:pathPrefix="/download"

      android:permission="android.permission.INTERNET" />

      <grant-uri-permission android:pathPrefix="/all_downloads/" />

      </provider>

 

     <service

      android:name="com.eebbk.bfc.component.download.service.DownloadService"/>

 

     <receiver

     android:name="com.eebbk.bfc.component.download.service.DownloadReceiver" >

     <intent-filter>

      <action android:name="android.intent.action.BOOT_COMPLETED" />

      <action android:name="android.net.conn.CONNECTIVITY_CHANGE" />

      </intent-filter>

      <intent-filter>

      <action android:name="android.intent.action.MEDIA_MOUNTED" />

      <data android:scheme="file" />

      </intent-filter>

      </receiver>

<!-- Download system End -->

 

4、使用DownloadController（下载控制器），进行下载任务的添加、删掉、更新、监听等操作，具体接口在SDK文档中有详细描述。
